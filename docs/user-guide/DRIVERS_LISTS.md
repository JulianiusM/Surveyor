# Drivers Lists Guide

Coordinate transportation and carpooling for events.

## Quick Start

1. **Create List**: Event name and date
2. **Add Drivers**: Name, car, seats
3. **Assign Passengers**: Who rides with whom
4. **Share**: Coordinate pickup details
5. **Track**: Monitor assignments

## Creating a Drivers List

### Basic Information
- **List Name**: Event or trip name
- **Event Date**: When
- **Details**: Location, time, notes

### Adding Drivers
- **Driver Name**: Who's driving
- **Vehicle**: Car description
- **Capacity**: Number of seats
- **Contact**: Phone/email
- **Notes**: Special info

## Managing Drivers

### Driver Information
- Name and contact
- Vehicle details
- Seat capacity
- Availability
- Pickup locations

### Editing Drivers
- Update information
- Change capacity
- Add/remove drivers
- Mark as unavailable

## Passenger Assignments

### Assigning Passengers
1. Select driver
2. Click "Assign Passenger"
3. Choose person
4. Confirm assignment
5. Track remaining seats

### Passenger View
Passengers can:
- See their driver
- View pickup details
- Contact driver
- Request changes

## Communication

### Sharing Details
- Driver contact info
- Meeting locations
- Pickup times
- Special instructions

### Coordination
- Send updates to all
- Message specific drivers
- Notify passengers
- Share logistics

## Best Practices

✅ **Do:**
- Collect driver info early
- Balance passenger loads
- Share contact details
- Confirm assignments
- Have backup plans

❌ **Don't:**
- Overload vehicles
- Skip contact info
- Change without notice
- Assume availability
- Forget insurance/licenses

## Tips

- Create list early
- Request drivers proactively
- Balance loads fairly
- Communicate clearly
- Have contingency drivers
- Share pickup locations
- Confirm day before
