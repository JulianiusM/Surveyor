# Drivers Lists Guide

Coordinate transportation for events by organizing drivers and passengers.

## What are Drivers Lists?

Drivers lists help you:
- Register available drivers
- Specify passenger capacity
- Let participants sign up for rides
- Track who's riding with whom
- Support guest participation

## Creating a Drivers List

### Basic Steps

1. **Dashboard** → Drivers Lists → "+ Create"
2. **Fill in:**
   - **Title** (required) - e.g., "Conference Transportation"
   - **Description** (optional) - Meeting location, time, etc.
   - **Assign to Event** (optional) - Link to an event
3. **Set Permissions** - Who can view/participate
4. **Create List**

---

## Adding Yourself as a Driver

### Quick Add Form

At the bottom of the drivers list page:

1. **Driver Name** - Auto-filled with your name
2. **Description** - Vehicle info, meeting point, special notes (optional)
3. **Max # (capacity)** - How many passengers (default: 1)
4. **Click "Add"**

You appear in the drivers table immediately.

### What to Include in Description

Use the description field to specify:
- Vehicle type or color (e.g., "Blue Toyota")
- Meeting point (e.g., "South parking lot")
- Departure time
- Route or destination notes
- Any restrictions or preferences

**Example:** "Blue Honda, leaving from Main St at 8am, going via Highway 5"

## Signing Up as Passenger

### Finding a Ride

1. **View Drivers List** page
2. **Review Available Drivers:**
   - Driver name
   - Description (vehicle, meeting point, etc.)
   - Current passengers (Assignees column)
   - Available seats (Assigned / Max column)

3. **Choose a Driver:**
   - Find driver with open seats
   - Check description matches your needs
   - Click "Take" button

4. **You're Assigned:**
   - Your name appears in Assignees list
   - Seat count updates
   - If full, "Full" badge appears

### Removing Yourself

To cancel your ride:
1. Find the driver you're assigned to
2. Click "Remove" button next to your ride
3. Your name is removed
4. Seat becomes available again

## Managing the List (Owners)

### Editing Driver Details

If you have edit permissions:
- **Double-click Driver name** to edit
- **Double-click Description** to update vehicle/meeting info
- **Double-click Max capacity** to change passenger limit
- Changes save automatically

### Removing Passengers

Owners can remove any passenger:
- Click **× button** next to passenger name in Assignees column
- Person is unassigned immediately

### Reordering Drivers

Drag drivers to reorder:
- Click and hold on a driver row
- Drag to new position
- Release to drop
- Useful for grouping by departure time or location

### Deleting Drivers

If permitted:
- Click **trash icon** on driver row
- Confirm deletion
- Driver and all passenger assignments removed

## Guest Participation

### As a Guest Driver

1. **Receive Link** from organizer
2. **Register as Guest** (name + optional email)
3. **Save Personalized Link** to access again
4. **Add Yourself as Driver** using quick form

### As a Guest Passenger

Same registration process, then:
- Click "Take" on available driver
- Your name shows in passenger list

### Guest Capabilities

Guests can:
- ✅ Add themselves as drivers
- ✅ Sign up as passengers
- ✅ Remove themselves
- ✅ See all drivers and passengers

Guests cannot:
- ❌ Edit other drivers' info
- ❌ Remove other passengers (unless owner permission)
- ❌ Delete the list

## Understanding the Display

### List Statistics

At the top:
- **Participants** - Total people involved (drivers + passengers)
- **Drivers** - Number of driver entries
- **Open** - Drivers with available seats
- **Empty** - Drivers with no passengers yet

### Driver Table

Each row shows:
- **Driver** - Person offering ride
- **Description** - Vehicle and logistics info
- **Assignees** - List of passengers
- **Assigned / Max** - Current vs. capacity (e.g., "2 / 4")
- **Action** - Take/Remove buttons

### Status Indicators

- **"Open"** - Has available seats
- **"Full"** badge - At capacity
- **"Empty"** - No passengers assigned yet

## Tips and Best Practices

✅ **For Drivers:**
- Provide clear vehicle description
- Specify meeting location and time
- Update description if plans change
- Confirm with passengers before departure
- Include contact info in description if needed

✅ **For Passengers:**
- Check driver description carefully
- Confirm meeting point and time
- Contact driver if questions
- Remove yourself early if plans change
- Communicate with your driver

✅ **For Organizers:**
- Encourage drivers to register early
- Verify total capacity vs. expected participants
- Coordinate meeting points
- Share list link with all participants
- Follow up to ensure everyone has a ride
