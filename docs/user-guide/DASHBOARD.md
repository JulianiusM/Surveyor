# Dashboard Guide

Your dashboard is your personal homepage showing all the surveys, packing lists, activity plans, drivers lists, and events you've created.

## Dashboard Overview

After logging in, you'll see:

### Welcome Header
- Your display name
- Admin badge (if applicable)

### Entity Sections (Accordion Style)

Click each section to expand/collapse:

- **📊 Your Surveys** - Surveys you've created
- **📦 Your Packing Lists** - Packing lists you own
- **📅 Your Activity Plans** - Activity schedules you manage
- **🚗 Your Drivers Lists** - Transportation coordination lists
- **🎉 Your Events** - Events you're organizing (if managing events)

Each section shows:
- Entity title (clickable to view)
- Quick information (dates, counts)
- Creation date
- Action buttons

### Creating New Items

Each section has a "+ Create" button at the end of the search field:
- Click to open the creation form
- Fill in all required information
- After creating, you'll be taken to the new entity's page

## Empty State

If you haven't created any items yet:
- Each section shows an empty message when expanded
- A "+ Create" button at the end of the search field
- Helpful icon illustration

This is normal for new users - start creating by using the create button!

## Using Your Dashboard

### Viewing Items

Click any entity title to open its full page with all details and management options.

### Quick Access

Your dashboard provides quick access to:
- All your surveys for collecting votes
- Packing lists for trip coordination
- Activity plans for scheduling
- Drivers lists for transportation
- Events you're managing

### Managing Items

From the full entity page, you can:
- Edit details
- Manage permissions
- Share with others
- Delete (when appropriate)

## Navigation

- **Top Menu** - Quick links and user profile
- **Accordion Sections** - Click headers to expand/collapse
- **Entity Links** - Click titles to view full pages
- **Create Buttons** - Start new entities

## Tips

✅ **Organization:**
- Use descriptive titles for easy identification
- Archive or delete old items you no longer need
- Review dashboard regularly

✅ **Quick Actions:**
- Expand multiple sections to see all at once
- Collapse sections you're not currently using
- Click entity titles to jump directly to management
