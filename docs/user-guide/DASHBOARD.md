## Dashboard Overview

### What You'll See

When you log in, your dashboard displays:

1. **Welcome Header**
   - Your name or username
   - Admin badge (if applicable)

2. **Entity Sections**
   - Surveys
   - Packing Lists
   - Activity Plans
   - Drivers Lists
   - Events (if organizing)

3. **Quick Actions**
   - Create buttons for each entity type
   - Access to frequently used features

## Understanding Sections

### Accordion Sections

Each entity type has its own collapsible section:

- **Click section header** to expand/collapse
- **See count** of entities in each section
- **Empty state** shows when no entities exist

### Section Structure

Each section shows:

1. **Header**
   - Icon representing entity type
   - Title (e.g., "Your Surveys")
   - Entity count
   - Create button

2. **Entity List**
   - Entity titles (clickable)
   - Quick info (date, status)
   - Action buttons

3. **Empty State**
   - Helpful message when no entities
   - Large create button
   - Icon illustration

### Entity Types

#### 📊 Surveys
- Your created surveys
- Survey title
- Response count
- Creation date

#### 📦 Packing Lists
- Your packing lists
- List name
- Item count
- Completion status

#### 📅 Activity Plans
- Your activity schedules
- Plan name
- Date range
- Slot count

#### 🚗 Drivers Lists
- Your drivers lists
- List name
- Driver count
- Date/event

#### 🎉 Events (if applicable)
- Events you're organizing
- Event name
- Registration count
- Event date

## Creating New Items

### Quick Create

From dashboard sections:

1. **Click "+ Create" button** in section header
2. **Or click large button** in empty section
3. **Fill in creation form**
4. **Submit to create**

### Create Menu (Top Navigation)

Alternative creation method:

1. **Click "Create" in top menu**
2. **Select entity type** from dropdown
3. **Complete creation form**
4. **Return to dashboard** after creation

## Managing Entities

### Viewing Entities

**From Dashboard:**
- Click entity title to open
- See overview information
- Access full entity page

**Entity Card Shows:**
- Title/Name
- Key statistics
- Last modified date
- Quick actions

### Entity Actions

Available actions depend on entity type:

#### Common Actions
- **View**: Open full entity page
- **Edit**: Modify entity details
- **Share**: Manage permissions
- **Delete**: Remove entity (with confirmation)

#### Entity-Specific Actions
- **Surveys**: View responses, export results
- **Packing**: Mark items, assign tasks
- **Activities**: Manage slots, assign participants
- **Drivers**: Add drivers, assign passengers

### Sorting and Organization

Entities are typically sorted by:
- Most recently modified (default)
- Creation date
- Alphabetically by title

## Dashboard Tips

### Productivity Tips

✅ **Do:**
- Keep sections organized
- Delete unused entities
- Use descriptive titles
- Review dashboard regularly
- Collapse sections you don't need

### Efficiency Tips

1. **Keyboard Shortcuts**
   - Tab to navigate
   - Enter to open entity
   - Space to expand/collapse sections

2. **Bookmarking**
   - Bookmark dashboard page
   - Create browser shortcuts
   - Add to home screen (mobile)

3. **Regular Maintenance**
   - Archive old entities
   - Update entity titles
   - Clean up completed items

### Organization Strategies

**By Time:**
- Current projects at top
- Archive old ones
- Delete obsolete items

**By Importance:**
- Pin frequent items (if available)
- Keep active items visible
- Hide completed items

**By Type:**
- Group similar entities
- Use consistent naming
- Tag or categorize (if available)

## Troubleshooting

### Dashboard Won't Load

**Solutions:**
- Refresh page (Ctrl+R or Cmd+R)
- Clear browser cache
- Check internet connection
- Try different browser
- Contact support

### Entities Not Showing

**Possible Causes:**
- Permissions changed
- Entity deleted
- Filter applied
- Wrong account

**Solutions:**
- Verify you're logged in
- Check entity still exists
- Review permissions
- Contact entity owner

### Actions Not Working

**Solutions:**
- Check permissions
- Refresh page
- Clear browser cache
- Report bug
