# Activity Plans Guide

Schedule and coordinate group activities with time-slots and assignments.

## Quick Start

1. **Create Plan**: Name, dates, timezone
2. **Define Roles**: What positions needed
3. **Create Slots**: Time blocks with requirements
4. **Assign Participants**: Who does what when
5. **Auto-Assign**: Let system suggest assignments

## Creating an Activity Plan

### Basic Information
- **Plan Name**: Event or schedule name
- **Start/End Dates**: Duration
- **Timezone**: Local time
- **Description**: Activity details

### Defining Roles
- **Role Name**: Position (e.g., "Facilitator", "Helper")
- **Description**: Role responsibilities
- **Requirements**: Skills/qualifications

## Activity Slots

### Creating Slots
- **Time**: Start and end time
- **Title**: Activity name
- **Capacity**: How many participants
- **Roles Required**: Positions needed
- **Requirements**: Special needs

### Slot Management
- Add/edit/delete slots
- Duplicate for similar slots
- Bulk operations
- Conflict detection

## Participant Assignments

### Manual Assignment
1. Select slot
2. Click "Assign"
3. Choose participant
4. Select role
5. Confirm assignment

### Auto-Assignment
System suggests assignments based on:
- Participant availability
- Role requirements
- Fair distribution
- Preferences

### Taking Assignments
Participants can:
- View open slots
- Sign up for roles
- Leave assignments
- Update availability

## Advanced Features

### Requirements System
- **Skills**: Required abilities
- **Certifications**: Necessary qualifications
- **Experience**: Needed background
- **Overrides**: Manual exceptions

### Recommendations
- Smart suggestions
- Balanced scheduling
- Conflict resolution
- Optimization

### Availability Tracking
- Participants set availability
- Filter by available people
- Prevent overassignment
- Show conflicts

## Best Practices

✅ **Do:**
- Define roles clearly
- Set requirements early
- Use auto-assign when possible
- Balance assignments fairly
- Communicate changes

❌ **Don't:**
- Overassign people
- Ignore conflicts
- Skip role definitions
- Forget availability
- Change last minute
